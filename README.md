# Systems Observability & AI-Agent Sandbox
This repository serves as a testing ground for integrating enterprise systems (ServiceNow/ITSM) with AI-assisted automation.

## 🚀 The AI-Human Loop
In this project, I utilize **Claude 3.5** and **Cursor** to accelerate the development of automation scripts. I focus on:
- **Agentic Refactoring:** Using LLMs to deconstruct legacy scripts into modular, scalable Python functions.
- **Synthesized Documentation:** Generating technical SOPs directly from code logic to ensure a "Single Source of Truth."

## 🛠️ Tech Stack
- **Languages:** Python, Bash
- **Methodology:** ITIL 4, NIST Frameworks
- **AI Tooling:** Cursor IDE, Prompt Engineering (Chain of Thought)
